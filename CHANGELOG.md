# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-02-10

### Added
- **Project Launch**: Initial release of Mobile Automator (renamed from `mobile-qa-skills`).
- **TOML Architecture**: Lean command structure using namespaced TOML files (`/mobile:setup`).
- **Generator Pattern**: Dynamic workspace skill generation tailored to project tech stacks.
- **Mobile Automation**: Integration with `mobile-mcp` for device control (click, swipe, screenshot, etc.).
- **Frictionless Setup**: Integrated `npx` execution for automation tools.
- **Documentation**: Comprehensive guides in README.md, GEMINI.md, and CHANGELOG.md.
