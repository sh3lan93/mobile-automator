    Examples¶

    Real-world test scenario examples for Android and iOS.

    Available Examples¶

    • Android Examples — Android app test scenarios
    • iOS Examples — iOS app test scenarios

    Example Scenarios¶

    Explore complete test scenario examples covering: - Authentication flows (login, signup, logout) - App navigation and screen transitions - Form filling and data entry - List scrolling and filtering - Shopping cart and checkout flows - User profile editing - Settings configuration - Permission handling

    How to Use These Examples¶

    1. Review the scenario JSON to understand structure
    2. Adapt the scenario for your app
    3. Generate similar tests using /mobile-automator:generate
    4. Execute tests using /mobile-automator:execute
